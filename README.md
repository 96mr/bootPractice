## 쇼핑몰
> Spring boot 기반으로 진행한 개인 프로젝트입니다.


## 제작 기간
2023년 4월 10일 ~ (진행중)

## 사용 기술
__Back-end__
* Java 8
* Spring Boot 2.6
* Spring Security 5
* Spring-Data-JPA
* Oracle


__Front-end__
* HTML
* CSS
* JavaScript(JQuery)
* BootStrap

## 기능
* 회원
  * 회원가입
  * 로그인, 로그아웃
  * 회원 페이지 권한 체크
  * 배송지 등록, 변경
* 제품
  * 제품 등록
  * 제품 조회
  * 장바구니 담기
* 장바구니
  * 장바구니 조회
  * 제품 수량 변경, 삭제
* 주문
  * 장바구니 -> 주문
  * 바로구매 -> 주문
  * 주문 조회
* 리뷰
  * 리뷰 작성
  * 리뷰 조회
